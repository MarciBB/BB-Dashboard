## Requisiti

- Python 3.10+ (consigliato)
- Pip / venv

## Setup locale

```bash
# clona il repo
git clone https://github.com/<TUO-USERNAME>/BB-Dashboard.git
cd BB-Dashboard

# (opzionale) crea venv
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate

# installa dipendenze
pip install -r requirements.txt
